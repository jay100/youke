# 有课在线系统！

> 相关技术框架 electron + vue.js + element-ui 、 ffmpeg、阿里云直播、消息队列 MQ

## 安装步骤

``` bash

快速开始

npm install

或者直接下载安装包

https://github.com/jay100/youke/releases

运行

npm run start

```
